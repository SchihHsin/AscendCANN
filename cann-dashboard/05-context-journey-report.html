Scoring Method

Agentic 评分结果

评分区间为 0-100。分数越高，说明 agent 越能在少人工干预的情况下自主推进。

阶段分公式
组成项满分判断标准
文档清晰度30动作链、版本、路径、成功判据是否清楚。
环境可达性30常见开发环境能推进到哪一步，前置限制越少越高。
自动化友好度25是否容易脚本化推进，是否依赖手工复制、跨仓拼装。
反馈与验证质量15是否有 help、日志、测试、benchmark、结果校验、失败反馈。

阶段分数 = 文档清晰度 + 环境可达性 + 自动化友好度 + 反馈与验证质量

场景分公式

场景总分按本场景所包含阶段的权重重新归一化计算，不做简单平均。

场景总分 = Σ(阶段分数 × 阶段权重) / Σ(场景已使用权重)

辅助证据

覆盖率与被引用仓库名只作为辅助证据，不作为主叙事。主叙事仍然是角色、场景和阶段得分。
