## 核心概念

- **DoubleBuffer** - Ping-Pong 缓冲区、计算与搬运并行
- **Tiling 策略** - 多核切分、尾核处理、负载均衡
- **流水线编排** - SetFlag/WaitFlag 事件同步
- **内存访问优化** - Bank 冲突、对齐、L2-Cache

---

## 优化决策树

```
数据量 < 8K 元素？
├── Yes → 基础实现即可
└── No → 需要优化
    ├── SIMD 矢量计算（必须）
    ├── Tiling 分块（必须）
    └── 性能要求高？
        ├── Yes → DoubleBuffer + 流水线
        └── No → SIMD + Tiling
```

---

## 实现要点

- ✅ 必须使用矢量 API
- ✅ 大数据必须实现 Tiling
- ✅ 高性能场景使用双缓冲
- ✅ 注意内存对齐和 Bank 冲突
- ✅ 参考官方优化模板
