# 复杂模式算子文档参考

## 判断依据

算子包含以下**复杂模式**之一：

| 模式类型 | 特征 | 典型算子 |
|----------|------|----------|
| **全局聚合** | `Σ`/`max`/`min` 覆盖整行/整列 | Softmax、ReduceSum、ReduceMax |
| **归一化** | 需要全局统计量（均值/方差） | LayerNorm、BatchNorm、GroupNorm |
| **注意力** | Softmax + 矩阵乘组合 | FlashAttention、CrossAttention |
| **多遍遍历** | 需要多次遍历同一数据 | 任何全局聚合操作 |

---

## 模式一：全局聚合

### 原理说明

当聚合操作（如 `Σ`、`max`、`min`、`any`、`all`）覆盖整行/整列时，**不能分段独立计算**。

### 错误示例 ❌

```cpp
// 分段独立计算 - 错误！
Segment 0: max0 = max(x[0:4095])
Segment 1: max1 = max(x[4096:8191])
// 每段用各自的 max 归一化 - 结果错误！
```

### 正确方法 ✅

```
Pass 1: 计算全局聚合值
  globalMax = max(max(x[0:4095]), max(x[4096:8191]))

Pass 2: 使用全局值计算中间结果
  exp_values = exp(x[0:8191] - globalMax)

Pass 3: 计算最终结果
  result = exp_values / sum(exp_values)
```

### 核心原则

> **全局聚合操作必须先算出全局聚合值，再用该值进行后续计算**

## 模式二：归一化

### 原理说明

归一化需要**全局统计量**（均值、方差），依赖全局聚合模式。

### 标准流程

```
Pass 1: 计算全局均值
  mean = sum(x) / n

Pass 2: 计算全局方差
  variance = sum((x - mean)²) / n

Pass 3: 归一化
  result = (x - mean) / sqrt(variance + eps)
```

## 模式三：注意力

### 原理说明

注意力 = **全局聚合（Softmax）** + **矩阵乘（Matmul）** 的组合模式。

### 标准流程

```
Pass 1: Q @ K^T  (矩阵乘)
  scores = Q @ K^T

Pass 2: Softmax (全局聚合)
  weights = softmax(scores)  # 每行全局归一化

Pass 3: weights @ V (矩阵乘)
  output = weights @ V
```

### 关键点

- Softmax 的全局聚合特性使得每行必须独立处理
- 大矩阵乘法需要结合 Tiling + 双缓冲优化

## 通用实现策略

### 判断是否为全局聚合

检查算子公式是否有以下特征：
- [ ] `Σ` 求和覆盖整行/整列
- [ ] `max/min` 覆盖整行/整列
- [ ] `any/all` 逻辑覆盖整行/整列
- [ ] 依赖全局均值/方差（归一化）

### 多遍遍历设计

1. **Pass 1**: 计算全局聚合值（max/sum/mean/variance）
2. **Pass 2**: 使用全局值计算中间结果
3. **Pass 3**: 计算最终结果

### 性能优化

- ✅ 复用中间结果，避免重复计算
- ✅ 合并遍历次数（如 Softmax 可两遍完成）
- ✅ 参考官方复杂算子示例

## 核心概念总结

- **全局聚合** - 跨段计算全局聚合值，多遍遍历
- **归一化** - 依赖全局统计量（均值/方差）
- **注意力** - Softmax + Matmul 组合模式

- ✅ 复用中间结果减少遍历次数
- ✅ 注意精度和数值稳定性
