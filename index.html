# CANN 推理示例
import acl
from cann import AscendContext

# 初始化 CANN 上下文
ctx = AscendContext(
  device_id=0,
  precision="fp16"
)

# 加载模型
model = ctx.load_model(
  "./resnet50.om"
)

# 执行推理
output = model.infer(input_data)
print(output.shape)

精心设计的工具链

从算子开发到大模型推理，CANN 提供完整的异构计算编程框架，让开发者专注于业务创新。

AscendCL 编程接口

统一的 C/C++ 编程接口，直接调用昇腾 NPU 的计算资源，实现极致性能。

TBE 算子开发

Tensor Boost Engine 提供 DSL 和 TIK 两种算子开发方式，轻松实现自定义算子。

MindSpore 深度集成

与 MindSpore 框架深度融合，同时支持 PyTorch / TensorFlow 等主流框架。

模型转换与优化

ATC 工具将训练模型转换为昇腾离线模型（.om），自动进行算子融合与优化。

性能调优工具

Profiling 工具链提供细粒度的 NPU 性能分析，快速定位性能瓶颈。

分布式训练支持

HCCL 通信库支持昇腾集群训练，助力千亿参数大模型分布式训练。

1. 调用 acl.init() 完成运行时初始化
2. 调用 acl.rt.set_device(device_id) 指定设备
3. 创建 Context 上下文以隔离执行环境
4. 创建 Stream 流以管理异步任务队列
